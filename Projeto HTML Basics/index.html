História do HTML

  1. HTML (HYPERTEXT MARKUP LANGUAGE) foi criado no final dos anos 80 por Tim Berners-Lee Como uma forma de compartilhar documentos entre os pesquisadores No ínicio o HTML era uma linguagem simples, com tags básicas, mas ele evoluiu rapidamente com o crescimento da internet ganhando mais tags e mais recursos.

  2. Em 1995 foi atualizado para o HTML 2.0 que adicionou formulários e tabelas e em 1999 foi implementado o HTML 4.01 onde ele se tornou o padrão da WEB trazendo mais elementos e suporte para folhas de estilo CSS

  3. Em 2014 foi criado então o HTML 5, implementando assim mais recursos como multimídia, elementos semânticos para melhor estruturação de conteúdo e APIs para interação com o navegador. Hoje o Html 5 é a base da WEB moderna sendo utilizado em conjunto com CSS e JavaScript para criar sites e aplicativos ricos e interativos.

    4. Tags e conceitos do HTML

    • As tags são blocos de contrução fundamentais no HTML elas são utilizadas para estruturar e dar significado ao conteúdo de uma página web, informando ao navegador de como deve ser apresentado o conteúdo.

    • Elementos:Um elemento HTML é formado por uma tag de abertura e uma de fechamento.

    • Elementos de Bloco e inline:Elementos de bloco ocupam toda largura disponível e forçam uma quebra de linha antes e depois, já os elementos inline ocupam apenas o espaço necessário e não forçam quebras de linha.

    • Aninhamento:Elementos HTML podem ser aninhados uns dentro dos outros para criar estruturas hierárquicas.

    Oque é o DOM?

    • Document Object Model (DOM) , ou Modelo de Documento de Objeto, é uma representação estruturada de um documento HTML ou XML. Ele permite que linguagens de programação, como JavaScript, acessem e modifiquem o conteúdo, a estrutura e o estilo de uma página web.

    • Em termos mais simples:

    • Imagine o DOM como uma árvore genealógica da sua página web. Cada elemento da página (títulos, parágrafos, imagens, links, etc.) é um "nó" nessa árvore. O DOM define como esses nós se relacionam entre si (pais, filhos, irmãos) e quais propriedades (texto, atributos, estilos) eles possuem.
